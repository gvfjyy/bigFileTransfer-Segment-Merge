# bigFileTransfer-Segment-Merge
大文件传输解决方案，切割→传输→合并。基于QT设计了一个界面。
## 基本介绍
c++11、文件缓冲区、二进制文件读取、写入。<br>
借鉴https://github.com/Jermmy/file_segmentation ，并对文件读写错误进行了修改，使之适用于所有类型文件。
## 使用方法
软件已打包至release.zip解压后可直接使用
