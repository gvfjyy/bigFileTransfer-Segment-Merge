# bigFileTransfer-Segment-Merge
大文件传输解决方案，切割→传输→合并。基于QT设计了一个界面。
## 基本介绍
c++11、文件缓冲区、二进制文件读取、写入。<br>
## 使用方法
软件已打包至release.zip解压后可直接使用<br>
1、分割：选择文件→执行→执行结束&生成json配置文件<br>
2、合并：选择json文件→执行→合并完成<br>
